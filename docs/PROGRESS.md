# Seedream 批处理应用开发进度跟踪

## 项目概述
基于 Seedream API 的图像批处理应用程序，使用 Gradio 框架开发，支持文生图、图生图、图像编辑和视频生成等功能，具备强大的批处理能力。

## 项目结构
```
SeedreamBatchProcess/
├── docs/               # 文档目录
│   ├── GUIDE.md       # 开发指南
│   ├── PROGRESS.md    # 进度跟踪（本文档）
│   └── README.md      # 用户说明
├── src/               # 源码目录
│   ├── ui/           # 用户界面模块
│   ├── api/          # API通信模块
│   ├── batch/        # 批处理模块
│   └── utils/        # 工具模块
├── config/           # 配置文件目录
├── input/            # 输入文件目录
├── output/           # 输出文件目录
├── logs/             # 日志文件目录
├── scripts/          # 脚本目录
├── requirements.txt  # Python依赖
└── main.py          # 主程序入口
```

## 技术栈
- **前后端框架**: Gradio
- **API客户端**: OpenAI Python SDK
- **图像处理**: Pillow (PIL)
- **配置管理**: JSON/YAML
- **日志系统**: Python logging
- **虚拟环境**: venv/conda

## 当前问题和风险
1. **API图像上传**: 本地图像文件需要转换为URL才能传递给API，目前暂时使用文件路径代替
2. **视频生成功能**: Seedream API的视频生成功能接口需要进一步确认
3. **并发处理**: 需要根据API限制调整并发数量
